Sistema RAG para estudio de Neurofisiología

Este proyecto implementa un sistema Retrieval-Augmented Generation (RAG) para el estudio académico de neurofisiología a partir de artículos científicos en formato PDF.

El sistema recupera fragmentos relevantes desde un índice vectorial (FAISS) y genera respuestas exclusivamente basadas en dichos documentos, sin incorporar conocimiento externo.

⚠️ Los documentos originales no se incluyen en el repositorio por motivos de copyright.
